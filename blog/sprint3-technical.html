The DOM

The Document Object Model.

The Document Object Model (or DOM) is what gives us the ability to interact with our web pages, and making us able to change existing properties, delete them and even add new ones.

Loops

What is a loop?

A loop is code that will continuously repeat an action until it meets a certain condition (or a goal). The order code is executed.

Functions

What are they and why are they useful?

Functions are kind of similar to loops in a way (not really but kind of), they are a subprogram you're writing out to also execute a certain task.
function name(parameters) { statements }
Above is an example of a function, "name" being whatever you're choosing to define the function as, parameters being the properties you want to use in the function and the statements are what you want them to do.
